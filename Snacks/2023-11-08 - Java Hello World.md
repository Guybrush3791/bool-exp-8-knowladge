## Repo
`hello-java`

## Todo
- creare il primo progetto Java chiamato `hello-java`
- aggiungere la classe `StampaSaluto` nel package `org.lessons.java`
- aggiungere il metodo `main` (a mano o sfruttando il context menu)
- nel metodo `main` aggiungere l’espressione per **stampare in console** un messaggio di **saluto**

Per condividere il repo su GitHub, come visto in classe e come indicato sulle slides, iniziamo ad usare il terminale:
- creare un file `.gitignore` contenente `*.class` (e `.DS_Store` per Mac)
- creare il repo su GitHub col nome `hello-java`
- posizionarsi col terminale nella directory del progetto java
- seguire la guida che compare alla creazione del progetto su **GitHub** per inizializzare e caricare il programma sulla repo (unica differenza e' peri il `git add` --> `git add -A`)

> [!attention] Se vi trovate in difficoltà o se vi viene qualche dubbio aprite il microfono
